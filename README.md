# Movie AI Search

Sistema de busca inteligente de filmes usando PostgreSQL, pgvector, FastAPI, Node.js, Railway, Neon e dados do TMDB.

## Projeto online

Site:
https://movie-ai-web-production.up.railway.app

API Python:
https://movie-ai-production.up.railway.app/health

## Tecnologias

- Python
- FastAPI
- Sentence Transformers
- Node.js
- Express
- PostgreSQL
- pgvector
- Neon
- Railway
- TMDB API

## Funcionalidades

- Busca semântica por filmes
- Busca por ator/atriz
- Busca por diretor
- Busca por tema, clima ou ideia
- Página de detalhes do filme
- Página de detalhes da pessoa
- Ranking por relevância, popularidade e intenção de busca
